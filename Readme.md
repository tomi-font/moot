## description

This:

- is a 2D video game engine written from scratch.
- uses the [ECS](https://en.wikipedia.org/wiki/Entity_component_system) paradigm.
- is not made with external audience in mind.

## dependencies

#### compiler

- Clang

#### libraries

- Boost.Mp11
- SFML 2.5.1
- Lua 5.4.6
- sol 3.3.0
